# /kd-kickoff — Resume Project

## Instructions
1. Read `{project-root}/KD_output/status/status.md` for project state
2. Read `{project-root}/.kracked/config/settings.json` for configuration
3. Read `{project-root}/.kracked/config/scale.json` for project scale
4. Read `{project-root}/.kracked/security/xp.json` for agent level
5. Read `{project-root}/.kracked/skills/memories/SKILL.md` for session memory

6. Greet user and summarize:
   - Project name and scale
   - Current stage
   - Last activity
   - Pending tasks
   - Agent level and XP

7. Ask: "Nak sambung dari sini atau ada arahan baru?"
