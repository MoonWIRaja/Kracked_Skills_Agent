# `/kd-kickoff`

## Purpose
Sambung semula projek sedia ada dan ringkaskan roster, stage, XP, serta next step.

## Required Actions
1. Baca status, XP, memories, dan roster.
2. Ringkaskan:
   - siapa main agent
   - siapa sub-agent penting
   - stage semasa
   - artefak terakhir
   - command seterusnya
3. Jika belum ada discovery, arahkan ke `/kd-analyze`.
