# `/kd-roster`

## Purpose
Paparkan roster utama projek dan handles `@Name` yang sah.

## Required Actions
1. Baca `.kracked/config/agents.json`.
2. Tunjukkan:
   - main agent
   - semua 11 sub-agent roles
   - nama semasa
   - mention handle
   - bila sesuai guna setiap agent
3. Terangkan bahawa:
   - user biasanya bercakap dengan main agent
   - direct specialist chat dibuat dengan `@Name`
   - multi-mention tidak disokong untuk fasa ini
4. Jika tiada roster, maklumkan install tidak lengkap.
