# `/kd-status`

## Purpose
Tunjukkan keadaan semasa projek, stage, XP, transcript/artifact terkini, dan next step.

## Required Actions
1. Semak status, XP, dan artifacts stage.
2. Ringkaskan:
   - current stage
   - last major artifact
   - current next command
   - main agent XP
   - latest consulted agents jika boleh dikenal pasti dari transcript
3. Jika ada gaps dalam flow, nyatakan dengan jelas.

## Footer
```text
Next command: /kd-help
XP updated: +5
Learning bonus: none
Memory updated: no
Artifacts written: [KD_output/status/status.md]
Agents consulted: [main-agent]
```
